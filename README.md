# 小米 MiMo 百亿 Token 激励计划 - 多智能体自动编程演示

## 项目简介
本项目使用 MiMo 大模型构建了一个多智能体协作编程系统，包含**架构师**、**程序员**、**测试员**和**修复员**四个 AI 角色。
它们通过长上下文对话和多次 API 调用，自动从零生成一个完整的待办事项 Web 应用，并完成代码审查与修复。

## 为什么这能展现 Token 消耗潜力？
- 每个角色的每次调用都需消耗数千 Token，且包含长上下文传递。
- 系统支持多轮迭代（审查→修复→再审查），Token 消耗可轻松扩展至数十万级别。
- 该框架可应用于更大型项目（如代码库重构），届时 Token 消耗将呈指数级增长。
